# Support CRM System

A modern full-stack Customer Support CRM System built using React, FastAPI, and NocoDB.

This application enables support teams to efficiently manage customer support tickets through a clean SaaS-style dashboard with ticket creation, search, filtering, status tracking, and notes/comments management.

Built as part of the Datastraw Assessment Test.
